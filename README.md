# MailPure.AI
AI-Powered Bot and DGA Email Detection System  /  AI Tabanlı Bot ve DGA E-posta Tespit Sistemi </br>
</br>
TR
</br>
Yapay zeka tabanlı bu sistem, e-posta adreslerinin yapısal özelliklerini (entropi, karakter dizilimi vb.) analiz ederek, adresin bir insan tarafından mı oluşturulduğunu yoksa algoritmalarla (DGA) üretilmiş bir bot hesabı mı olduğunu tespit eder.
</br>
</br>
EN
</br>

An AI-powered system that analyzes the structural patterns of email addresses (entropy, character distribution, etc.) to distinguish between human-generated identities and algorithmically generated (DGA) bot accounts.
